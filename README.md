# Self-Sovereign Identity (SSI) Platform

A blockchain-based decentralized identity management system.

## Features
- Decentralized Identifiers (DIDs)
- Verifiable Credentials
- Zero-Knowledge Proofs
- IPFS Storage
- Blockchain Anchoring

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Blockchain: Hyperledger Indy (Simulation)
- Storage: IPFS
